# ☕ Starbucks Sales Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes Starbucks beverage sales to provide insights into sales performance, beverage categories, store locations, and customer purchasing trends.

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Excel

## 📊 Dashboard Features
- Total Sales KPI
- Total Orders
- Average Calories
- Beverage Category Analysis
- Top Products
- Sales by Store
- Interactive Filters
- Dynamic Visualizations

## 📈 Key Insights
- Highest selling beverage categories
- Sales distribution by product type
- Customer preferences
- Performance trends
